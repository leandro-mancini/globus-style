Iniciando seu desenvolvimento com Globus Style

ngx-globus-style Um framework front-end de comportamento e estilo para projetos web, com uma interface usada por milhares de usuários. Mantido pelos desenvolvedores front-end da BgmRodotec.

Dependências

Angular npm install -g @angular/cli

Angular Material npm install --save @angular/material @angular/cdk.
Se estiver usando a versão do Angular 6 execute isto ng add @angular/material

Instalação:

Depois de instalar as dependências acima, instale o ngx-globus-style via npm:

npm install --save ngx-globus-style
